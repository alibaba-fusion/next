# Transfer

-   category: Components
-   family: DataEntry
-   chinese: 穿梭框
-   type: 基本

---

双栏穿梭选择框。

## 何时使用

-   用直观的方式在两栏中移动元素，完成选择行为。
-   需要在多个可选项中进行多选时。
-   比起 Select 和 TreeSelect，穿梭框占据更大的空间，可以展示可选项的更多信息。

## 无障碍键盘操作指南

| 按键         | 说明                        |
| :--------- | :------------------------ |
| Up Arrow   | 获取当前项的前一项焦点               |
| Down Arrow | 获取当前项的后一项焦点               |
| Enter      | 当前列表选中的项移动到另一个列表          |
| SPACE      | 选择/取消当前项或当前列表选中的项移动到另一个列表 |
