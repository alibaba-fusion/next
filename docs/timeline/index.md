-   category: Components
-   family: DataDisplay
-   chinese: 时间轴

## Guide

垂直展示的时间流信息。

### 何时使用

-   当有一系列信息需要从上至下按时间排列时。
-   需要有一条时间轴进行视觉上的串联时。
