# 只显示图标-宽度自适应

Nav 可设置 iconOnly 属性，只显示图标，以减少占用空间。通过设定 iconOnlyWidth 为 100%，可以实现 iconOnly 场景下 icon 自适应居中展示，需要注意的是这种情况下需要 Nav 外部容器有具体宽度。

# Only show icon - width adaptive
