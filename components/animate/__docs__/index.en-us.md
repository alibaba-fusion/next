### Animate

| Param           | Description                                                                                                                        | Type                                                              | Default Value | Required |

| beforeAppear    | Callback fired before the "entering" status of the first mount is applied<br/><br/>**signature**:<br/>**params**:<br/>_node_: Node | (node: HTMLElement) => void                                       | -             |          |
| onAppear        | Callback fired after the "entering" status of the first mount is applied<br/><br/>**signature**:<br/>**params**:<br/>_node_: Node  | (node: HTMLElement) => void                                       | -             |          |
| afterAppear     | Callback fired after the "entered" status of the first mount is applied<br/><br/>**signature**:<br/>**params**:<br/>_node_: Node   | (node: HTMLElement) => void                                       | -             |          |
| beforeEnter     | Callback fired before the "entering" status is applied<br/><br/>**signature**:<br/>**params**:<br/>_node_: Node                    | (node: HTMLElement) => void                                       | -             |          |
| onEnter         | Callback fired after the "entering" status is applied<br/><br/>**signature**:<br/>**params**:<br/>_node_: Node                     | (node: HTMLElement) => void                                       | -             |          |

| beforeLeave     | Callback fired before the "exiting" status is applied<br/><br/>**signature**:<br/>**params**:<br/>_node_: Node                     | (node: HTMLElement) => void                                       | -             |          |
| onLeave         | Callback fired after the leave stage<br/><br/>**signature**:<br/>**params**:<br/>_node_: Node                                      | (node: HTMLElement) => void                                       | -             |          |
